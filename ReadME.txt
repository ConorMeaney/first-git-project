Galileo: I cover alot of this in the word doc. 
You need to use specfically the arduino 1.5.3 IDE to run the IntelGalileoCode. 
Once you connect the intel galileo to the Pc it will need to update a drive.
Once the galileo is connect you need to make sure the IDE is set to the correct board in the tools tab.
The galileo needs to be on the right com port by going into tools and scrolling down to serial port.
